Reflexão e Aplicação Prática

Propriedades CSS utilizadas

  • display: flex; — Define o container como flexível;
  • flex-direction — Controla a direção dos itens (linha, coluna);
  • justify-content — Alinha os itens ao longo do eixo principal;
  • align-items — Alinha os itens ao longo do eixo transversal;
  • flex-wrap — Permite que os itens quebrem a linha e vão para a próxima;
  • order — Define a ordem visual dos itens;
  • align-self — Alinha individualmente um item;

Aplicação prática imaginada

Por exemplo: Para construir um menu de navegação responsivo, usaria Flexbox para alinhar os itens horizontalmente com justify-content: space-between;, garantindo que o menu se adapte bem em diferentes tamanhos de tela. Também usaria flex-wrap para permitir que os itens se movam para a próxima linha em telas pequenas. Além disso, para uma galeria de fotos, organizaria as imagens em uma grade flexível com espaçamento e alinhamento adequados para manter a estética.
